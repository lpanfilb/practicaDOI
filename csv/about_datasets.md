# Main dataset

- The main dataset is `oscar_award.csv`.
- It contains data about the past oscar nominees between 1927 and 2025.
- URL: https://www.kaggle.com/datasets/unanimad/the-oscar-award

# Other datasets

- Another dataset we found is ``oscar_demographics.csv``.
- It contains more data about oscar winners since 1928.
- URL: https://www.kaggle.com/datasets/fmejia21/demographics-of-academy-awards-oscars-winners/data
